# Demo instructions

1. Unzip this bundle of files. (hooray you've gotten this far!)
2. Open {{PRODUCTION_URL}} in your browser.
3. In the app, follow the on-screen instructions to upload these files (step 2).
4. None of the files are password-protected, so no password is needed.
5. Tick the acknowledgement checkbox.
6. Click “Run report” to generate the output.
